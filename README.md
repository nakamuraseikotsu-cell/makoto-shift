# シフト最適化 Web パネル

5 院（国分寺・武蔵小金井・東小金井・小金井坂下・人形町）のシフト最適化を
Streamlit で可視化するアプリです。

- **メインスクリプト**: `streamlit_app.py`
- **最適化エンジン**: `shift_optimizer/`
- **デプロイ手順**: [`DEPLOY.md`](./DEPLOY.md)

## 🔑 ログイン

テスト公開中はパスワードが必要です（運用担当者から共有されたパスワードを入力）。

## 🚀 ローカルで実行

```powershell
pip install -r requirements.txt
streamlit run streamlit_app.py
```
